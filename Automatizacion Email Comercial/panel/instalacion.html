Instalación

Módulo independiente del chatbot (puerto 8020 vs 8000).

1. Arrancar en local

Doble clic en start-email-comercial.command o:

cd "Automatizacion Email Comercial/backend"
python3 -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt
cp .env.example .env   # añade ANTHROPIC_API_KEY
python run.py

Panel: http://127.0.0.1:8020/panel/

2. Webhook (Make / n8n)

POST /empresas/{empresa_id}/draft

{
  "lead_name": "María García",
  "lead_company": "Clínica Ejemplo SL",
  "sales_rep_name": "Carlos Ruiz",
  "incoming_email_body": "Texto del correo del lead…",
  "email_subject": "Consulta integración"
}

La respuesta incluye analysis y email_draft. Nunca envía correo al cliente.

3. Cumplimiento

  • Human-in-the-loop: solo borradores; el comercial revisa y envía.
  • Anti-alucinación: si no está en la documentación, placeholder explícito en el cuerpo.
  • RGPD: minimización de datos (nombre, empresa, contexto técnico).
