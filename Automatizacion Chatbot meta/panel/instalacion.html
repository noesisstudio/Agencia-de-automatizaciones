Guía de instalación

30–45 min la primera vez (Meta incluido). Sigue los pasos en orden.

  1. 1
    Arrancar la API
    cd chatbot-backend
source .venv/bin/activate
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000
    Abre http://127.0.0.1:8000/panel/
  2. 2
    Clave de IA
    En .env pon ANTHROPIC_API_KEY. Reinicia uvicorn.
  3. 3
    Crear un cliente
    Clientes → ID (ej. clinica-norte) y nombre.
  4. 4
    Documentos
    Markdown/PDF → Publicar chatbot.
  5. 5
    Probar chat
    Probar conversación con diagnóstico integrado.
  6. 6
    WhatsApp
    WhatsApp → Phone number ID + mensaje de prueba.

Archivo .env mínimo

ANTHROPIC_API_KEY=sk-ant-...
META_ACCESS_TOKEN=
META_VERIFY_TOKEN=openix_verify_token_seguro
META_APP_SECRET=
CHATBOT_TENANT_DEFAULT=mi-empresa

Guía Meta WhatsApp

META_WHATSAPP.md — variables, webhook y envío de prueba.

Problemas frecuentes

  • Bot no responde — Probar → Ver diagnóstico
  • WhatsApp no contesta — webhook /whatsapp/webhook + ngrok
  • Panel no carga — uvicorn en puerto 8000
