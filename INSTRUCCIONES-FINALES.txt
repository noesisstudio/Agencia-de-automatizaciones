PASO 1: Ve a https://railway.app/dashboard

PASO 2: Haz clic en "New Project" → "Deploy from GitHub"

PASO 3: Conecta tu repositorio:
  • GitHub repo: noesisstudio/Agencia-de-automatizaciones
  • Root Directory: Chatbot pagina web/bot-b/backend

PASO 4: Configura variables de entorno (importante):
  ANTHROPIC_API_KEY = sk-ant-v1-... (tu clave)
  TENANT_ID = noesis
  EMPRESA_NOMBRE = Noesis
  APP_ENV = production
  ALLOWED_ORIGINS = https://noesis.com,https://www.noesis.com

PASO 5: Espera a que termine de desplegar (2-3 minutos)

PASO 6: Copia la URL que aparece en la esquina superior
  Ejemplo: https://chatbot-noesis-prod.railway.app
  ⚠️  LA NECESITARÁS DESPUÉS

PASO 1: En Netlify, ve a tu sitio

PASO 2: Site settings → Build & deploy → Environment variables

PASO 3: Añade una variable:

PASO 4: Guarda

PASO 5: Fuerza un redeploy:
  Site settings → Trigger deploy → Deploy site
