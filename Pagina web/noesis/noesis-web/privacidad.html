Privacidad

Cómo tratamos los datos.

Esta política explica qué datos puede tratar Noesis cuando una persona solicita información, usa el formulario de diagnóstico, accede al área privada de cliente o interactúa con nuestro chatbot con IA.

Responsable

Noesis. Contacto para privacidad, derechos y consultas sobre tratamiento de datos: info@bynoesis.com.

Datos que podemos tratar

  • Nombre, email, teléfono, empresa (formularios de contacto)
  • Mensajes enviados en formularios o chat
  • Conversación con el chatbot de IA (preguntas y respuestas)
  • Información técnica: dirección IP, navegador, hora de acceso
  • Datos de facturación cuando existe relación contractual

Chatbot con IA — Información Especial

Identificación como IA: El chatbot de Noesis es un sistema de inteligencia artificial. No es una persona. Cuando lo uses, estás interactuando con una máquina que procesa tus preguntas con Claude, un modelo de IA de Anthropic.

Qué datos procesamos:

  • Tu pregunta se envía a Claude API (Anthropic) para generar una respuesta
  • Guardamos el historial de tu conversación para mejorar el servicio
  • Tu sesión se identifica con un ID único almacenado en tu navegador (no en servidores externos)

Cómo procesamos tus datos en el chatbot:

  • Los datos se procesan bajo contrato de procesamiento de datos (DPA) con Anthropic
  • Claude no entrena ni aprende de tus datos individuales
  • Puedes eliminar tu historial de chat borrando los datos locales de tu navegador

Proveedores de IA:

  • Claude (Anthropic): Procesa las preguntas para generar respuestas. Cumple GDPR. Datos almacenados en servidores UE/EEUU con encriptación.

Finalidad

  • Responder solicitudes comerciales
  • Preparar diagnósticos de procesos
  • Gestionar servicios contratados
  • Responder preguntas frecuentes (chatbot)
  • Mejorar el chatbot y detectar problemas
  • Cumplir obligaciones legales

Base Legal

  • Consentimiento: Al usar el chatbot o enviar un formulario
  • Medidas precontractuales: Preparar diagnósticos y propuestas
  • Contrato: Cuando ya somos tu proveedor
  • Interés legítimo: Mejorar nuestros servicios y detectar fraude
  • Obligación legal: Cumplir normativa fiscal y de protección

Conservación

Los datos se conservarán durante el tiempo necesario para atender la solicitud, mantener la relación comercial o cumplir obligaciones legales (hasta 6 años para datos fiscales).

Destinatarios

Noesis no venderá datos personales. Podrán existir proveedores técnicos necesarios para:

  • Alojar la web y correo (servidores europeos)
  • Procesar preguntas con IA (Anthropic - Claude)
  • Almacenar conversaciones de forma segura

Todos los proveedores cumplen GDPR y tienen cláusulas de procesamiento de datos.

Derechos

Puedes ejercer los siguientes derechos escribiendo a info@bynoesis.com:

  • Acceso: Ver qué datos tenemos sobre ti
  • Rectificación: Corregir datos incorrectos
  • Supresión: Eliminar tus datos (derecho al olvido)
  • Limitación: Restringir cómo usamos tus datos
  • Portabilidad: Recibir tus datos en formato exportable
  • Oposición: Oponerme al procesamiento de tus datos

También puedes reclamar ante la autoridad de protección de datos de tu país.

Seguridad

Usamos encriptación (TLS/SSL) en tránsito. Los servidores están protegidos con cortafuegos, backups regulares y acceso restringido. Realizamos auditorías de seguridad periódicamente.

Cambios en esta política

Esta política puede cambiar ocasionalmente. Te notificaremos cambios significativos. La versión más actual siempre está disponible aquí.
