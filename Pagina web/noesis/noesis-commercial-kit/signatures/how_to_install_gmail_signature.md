# Com instal·lar la firma a Gmail

1. Obre Gmail amb el correu de Noesis.
2. Ves a `Configuració > Mostra tota la configuració`.
3. A l'apartat `Firma`, crea una firma nova.
4. Obre el fitxer `noesis_signature_miquel.html` al navegador.
5. Selecciona visualment la firma i copia-la.
6. Enganxa-la a Gmail.
7. Substitueix `[TELÈFON]` i `[WEB]`.
8. Desa els canvis.

## Recomanació

No posis una imatge massa gran a la firma. En fase inicial, millor una firma neta, ràpida i professional. Quan tinguem domini i web definitius, podem afegir el logo allotjat a la web.
