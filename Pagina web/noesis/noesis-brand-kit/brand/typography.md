# Noesis Typography

## Display

Primary choice: Newsreader.

Fallbacks: Georgia, serif.

Use for:

- Headlines.
- Brand statements.
- Logo wordmark.
- Editorial phrases.

## Body

Primary choice: Inter.

Fallbacks: Arial, sans-serif.

Use for:

- Paragraphs.
- Navigation.
- Buttons.
- Captions.
- Process cards.

## General rules

- Do not use negative letter spacing.
- Keep body text calm and readable.
- Reserve large serif type for strong editorial hierarchy.
- In compact UI elements, use Inter instead of the display serif.
