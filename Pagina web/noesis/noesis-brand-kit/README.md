# Noesis Brand Kit

Carpeta base d'identitat visual per Noesis.

## Estructura

- `logos/png`: logos exportats en PNG per ús directe.
- `logos/svg`: versions vectorials editables.
- `brand`: guia de marca, colors i tokens visuals.

## Ús ràpid

- Logo principal: `logos/png/noesis-logo-horizontal.png`
- Símbol principal: `logos/png/noesis-mark.png`
- Badge per avatar/favicon: `logos/png/noesis-mark-badge.png`
- Versió sobre fons fosc: `logos/png/noesis-logo-horizontal-light.png`

## Posicionament

Noesis és arquitectura operativa per empreses: ordre, traçabilitat i impacte mesurable en processos interns.
